# AR-Project-Year2

This is an Augmented reality experience project, that used a tracked marker image. This experience can be built to an android device. 

## the experience has:
- an explanation on the experience when loaded onto an android device. This can be disabled using the confirmation button.
- A 3D model that appears when hovered over a specific tracked marker image. The marker image is below:
